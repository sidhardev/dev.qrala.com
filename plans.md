# 3D Immersive Portfolio - Project Vision
## Dev Sidhar | Full-stack Developer

### 1. Overview
Ek immersive, 3D interactive showroom jo static portfolio ko ek digital experience mein convert karta hai. Is project ka goal Dev Sidhar ki technical expertise (Backend, Real-time systems, and UI) ko ek 3D space mein showcase karna hai.

---

### 2. The 3D Showroom Concept
* **Visual Style:** Ek sleek, dark-themed digital gallery ya showroom.
* **Environment:** User ek 3D hallway ya circular room mein navigate karega.
* **Interaction:** Har project ek dedicated "3D Pedestal" (stair-case model) par rakha hoga. Jab user scroll ya click karega, camera smooth-motion (GSAP) ke saath us project par zoom-in hoga.
* **Dynamic Lighting:** Real-time shadows aur neon accent lighting jo interactive elements ko highlight karti hai.

---

### 4. Technical Stack
Is site ko build karne ke liye modern standard tools ka use kiya jayega (No React):

* **3D Engine:** Three.js (for WebGL rendering and 3D environment).
* **Animations:** GSAP (GreenSock) + ScrollTrigger (for smooth camera paths and pedestal interaction).
* **Frontend UI:** Vanilla JavaScript, HTML5, aur Tailwind CSS (for overlays and menus).
* **Backend Engine:** PHP (for dynamic project data delivery, contact form handling, and API integration).
* **Models:** GLTF/GLB formats (optimised for web performance).
